# GoodHR 5

GoodHR 5 是新的云端控制台 + 本地执行器版本。

目标结构：

- `cloud/backend`：Go 云端 API，负责登录、任务元信息、配置、机器绑定和版本管理。
- `cloud/frontend`：Vue 云端页面，负责用户操作、任务编排、连接本地 Agent。
- `local-agent`：运行在用户电脑上的本地程序，负责 CloakBrowser、平台页面操作、截图、OCR 和本地任务数据。
- `docs`：架构说明、接口草案和任务进度。

第一版原则：

- 用户必须先登录云端。
- 云端页面自动探测 `127.0.0.1:55271-55279`。
- 候选人详情、截图、OCR 和招聘平台 cookie 只保存在本地。
- 云端只保存用户、配置、任务摘要和机器绑定信息。
