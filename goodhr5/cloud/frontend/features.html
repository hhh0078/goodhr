功能

GoodHR 能帮你做什么

围绕招聘打招呼流程，把重复点击、候选人初筛、任务统计和本地执行串起来。

AI 候选人筛选

根据岗位要求和候选人信息生成匹配分，支持详情建议分和打招呼建议分。

关键词筛选

支持关键词、排除词、AND/OR 匹配，适合规则明确的岗位。

平台账号管理

通过本地浏览器扫码登录招聘平台，cookie 加密保存，团队内可共享。

本地 Agent

浏览器控制、截图、OCR、声音提醒和 cookie 解密都在本地完成。

任务日志

任务运行过程可展开查看，方便定位本地连接、AI 配置、cookie 或平台页面问题。

自动聊天跟进

候选人回复后，系统可根据岗位信息继续沟通，帮助确认意向、经验和到岗情况。

邀约面试

对高匹配、高意向候选人继续推进面试邀约，让打招呼不只停在第一句话。

永久免费功能

关键词筛选长期免费，适合先用规则跑招聘任务；AI 筛选、自动跟进和邀约面试可按需升级。

会员和邀请

支持订阅套餐、激活码兑换和邀请奖励，适合个人和小团队使用。
