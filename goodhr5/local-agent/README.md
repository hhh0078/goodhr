# GoodHR 5 Local Agent

本地 Agent 运行在用户电脑上，后续负责：

- CloakBrowser 控制。
- 招聘平台页面查找、滚动和点击。
- 当前可见候选人提取。
- 详情弹框截图。
- OCR。
- 本地任务 JSON、截图和 OCR 文件管理。

当前版本只提供 `/health`，用于云端页面探测。

## 启动

```bash
python3 -m app.main
```

默认从 `127.0.0.1:9001` 到 `127.0.0.1:9009` 自动尝试，遇到端口占用会继续尝试下一个端口。

如果设置了 `GOODHR_AGENT_PORT`，会优先尝试该端口，然后继续尝试默认端口范围。
