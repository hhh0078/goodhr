# GoodHR Local Agent 打包说明

本文说明如何把 `goodhr5/local-agent` 打包成可双击运行的桌面程序。

## 一、打包产物

macOS 打包后生成：

```text
dist/GoodHR.app
```

Windows 打包后生成：

```text
dist/GoodHR/
```

Windows 安装器打包后生成：

```text
dist-installer/GoodHRSetup-版本号.exe
```

## 二、运行数据目录

用户双击运行后，程序数据不会写入安装包内部，而是写入系统应用目录。

macOS：

```text
~/Library/Application Support/GoodHR/
```

Windows：

```text
%LOCALAPPDATA%/GoodHR/
```

目录用途：

```text
agent_data/   本地任务、机器码、绑定信息
cookies/      浏览器登录 profile
config/       配置文件
vendor/       CloakBrowser 运行文件
```

每次双击启动时，程序只清空窗口中的本次运行日志，不再写入 `logs/agent.log`。

## 三、推荐一键打包

macOS 在本机执行：

```bash
cd /Users/Zhuanz/Downloads/goodHR/goodhr5/local-agent
sh packaging/build_mac_full.sh
```

Windows 在 Windows 电脑上执行：

```bat
cd goodhr5\local-agent
packaging\build_windows_full.bat
```

Windows 安装器在 Windows 电脑上执行：

```bat
cd goodhr5\local-agent
packaging\build_windows_installer.bat
```

这两个脚本会自动：

1. 创建 `.venv`。
2. 配置 pip 阿里云镜像。
3. 安装运行和打包依赖。
4. 执行 PyInstaller 打包。
5. 首次运行打包产物时，程序会自动下载 CloakBrowser 并创建桌面快捷方式。
6. 安装器脚本会在文件夹版打包完成后调用 Inno Setup 生成安装包。

## 四、手动准备 Python 环境

进入 local-agent 目录：

```bash
cd /Users/Zhuanz/Downloads/goodHR/goodhr5/local-agent
```

创建虚拟环境：

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -U pip
```

安装运行和打包依赖：

```bash
pip install -e ".[packaging]"
```

## 五、CloakBrowser 下载配置

打包产物不再内置 CloakBrowser 压缩包。首次运行时，程序会请求：

```text
https://goodhr5.58it.cn/agent-browser-downloads.json
```

该 JSON 返回当前平台的浏览器下载地址：

```json
{
  "mac": "macOS 浏览器压缩包地址",
  "win": "Windows 浏览器压缩包地址"
}
```

下载后的文件保存在运行数据目录：

```text
vendor/downloads/
```

解压后的浏览器保存在：

```text
vendor/cloakbrowser/
```

第二次启动会直接复用已解压的浏览器，不会重复下载。

## 六、macOS 手动打包

在 macOS 上执行：

```bash
cd /Users/Zhuanz/Downloads/goodHR/goodhr5/local-agent
source .venv/bin/activate
sh packaging/build_mac.sh
```

脚本会做这些事：

1. 用 PyInstaller 打包：

```text
dist/GoodHR.app
```

## 七、Windows 手动打包

Windows 版请在 Windows 电脑上执行：

```powershell
cd goodhr5\local-agent
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install -U pip
pip install -e ".[packaging]"
powershell -ExecutionPolicy Bypass -File packaging\build_windows.ps1
```

脚本会做这些事：

1. 用 PyInstaller 打包：

```text
dist\GoodHR
```

## 八、注意事项

- macOS 只能可靠打包 macOS app。
- Windows exe 建议在 Windows 电脑上打包。
- 推荐优先使用 `packaging/build_mac_full.sh`、`packaging/build_windows_full.bat` 和 `packaging/build_windows_installer.bat`。
- 生成 Windows 安装器前，需要先安装 Inno Setup 6。
- `vendor/cloakbrowser/`、`vendor/downloads/`、`build/`、`dist/` 不提交 Git。
- 如果更新 CloakBrowser 下载地址，只需要更新前端公开 JSON 后重新部署官网。
